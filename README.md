# Corona
App simples consumindo [API](https://covid19.mathdro.id/api/) do covid19

### Tech
Tecnologias usadas no app
* [React Native]
* [Expo]
  
### Installation

#### Clone o repositório
[corona](https://github.com/TheNeoCarvalho/corona.git)

#### Instalar as dependencias do projeto.

```sh
$ cd corona
$ npm i 
ou 
$ yarn
```
### Rodar a aplicação

```sh
$ npm start 
ou
$ yarn start
```
